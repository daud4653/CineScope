# Install AI Dependencies

## Quick Install (If you have Python 3.9-3.11)

```bash
pip install tensorflow numpy
```

## Check Python Version

```bash
python --version
```

**Required:** Python 3.9, 3.10, or 3.11 (TensorFlow doesn't support 3.14 yet)

## Current Status

✅ **Server works fine without TensorFlow** - Uses fallback recommendations
✅ **No crashes** - All errors handled gracefully
⚠️ **AI recommendations use popular movies** instead of ML model

## To Enable Full AI Recommendations

1. Install Python 3.9-3.11
2. Run: `pip install tensorflow numpy`
3. Restart server
4. AI recommendations will work automatically
